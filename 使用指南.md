# 🚀 快速使用指南

## 安装应用
1. 找到 `release/AI插件应用 Setup 1.0.0.exe` 文件
2. 双击运行安装程序
3. 选择安装路径，点击安装
4. 安装完成后在桌面或开始菜单打开应用

## 首次配置
1. 打开应用后，点击左下角的 **⚙️ 设置** 按钮
2. 在"OpenAI API Key"输入框中粘贴你的API Key
3. 点击 **保存**

## 获取OpenAI API Key
1. 访问 [OpenAI Platform](https://platform.openai.com)
2. 注册或登录账号
3. 点击左侧菜单的 "API Keys"
4. 点击 "Create new secret key"
5. 复制生成的Key（格式：sk-...）
6. 粘贴到应用的设置中

## 使用场景

### 📚 学生学习
1. 点击左侧边栏的 **📖 学习辅导**
2. 输入你的问题，例如：
   - "帮我解释一下勾股定理"
   - "这道数学题怎么做：2x + 5 = 15"
   - "帮我写一篇关于环境保护的论文大纲"

### 💼 办公效率
1. 点击左侧边栏的 **💼 办公效率**
2. 输入你的需求，例如：
   - "帮我写一封商务邮件，内容是..."
   - "帮我总结这份报告的要点"
   - "帮我制作一个项目计划表"

### 💻 编程开发
1. 点击左侧边栏的 **💻 编程开发**
2. 输入你的问题，例如：
   - "用Python写一个爬虫程序"
   - "这段代码有什么bug？"
   - "帮我优化这段SQL查询"

### 🎨 图像生成
1. 点击左侧边栏的 **🎨 图像生成**
2. 描述你想要的图片，例如：
   - "一只在太空中飞翔的猫"
   - "未来城市的夜景"
   - "中国山水画风格的日落"
3. 点击 **🎨 生成** 按钮

### 📎 文件处理
1. 点击输入框左侧的 **📎** 按钮
2. 选择要上传的文件（支持PDF、TXT、图片）
3. 输入你的问题，例如：
   - "总结这份文档的要点"
   - "这张图片里有什么？"
   - "翻译这份文档"

## 快捷操作
- **Enter键** - 发送消息
- **清空对话** - 点击左下角的 🗑️ 按钮
- **切换场景** - 点击左侧边栏的场景按钮

## 常见问题

### Q: API Key安全吗？
A: 非常安全！API Key仅保存在你电脑的本地存储中，不会上传到任何服务器。

### Q: 使用需要付费吗？
A: 应用本身免费，但OpenAI API会按使用量收费。具体价格参考OpenAI官网。

### Q: 支持哪些文件类型？
A: 目前支持：PDF、TXT、PNG、JPG、JPEG、GIF。后续会支持Word、Excel等。

### Q: 可以离线使用吗？
A: AI对话功能需要联网。未来会推出离线模式（使用本地AI模型）。

### Q: 图像生成失败怎么办？
A: 检查API Key是否正确，确保你的OpenAI账号有DALL-E的使用权限。
