## 🚀 快速开始

### 方式一：使用Ollama本地模型（推荐）

#### 1. 安装Ollama
访问 [https://ollama.com](https://ollama.com) 下载并安装

#### 2. 下载模型
```bash
# 下载DeepSeek R1
ollama pull deepseek-r1

# 下载通义千问
ollama pull qwen2.5
```

#### 3. 启动Ollama服务
```bash
ollama serve
```

#### 4. 打开AI助手应用
- 应用会自动连接到 `http://localhost:11434`
- 在设置中选择要使用的模型
- 开始对话！

### 方式二：使用OpenAI API

1. 打开应用，点击左下角 **⚙️ 设置**
2. 选择 **☁️ OpenAI API** 引擎
3. 输入你的API Key
4. 点击保存

## ⚠️ 注意事项

1. **Ollama使用**：
   - 必须先安装并启动Ollama服务
   - 确保已下载所需模型
   - 默认地址为 `http://localhost:11434`

2. **API Key安全**：
   - 所有配置仅保存在本地
   - 不会上传到任何服务器

3. **图像功能**：
   - 图像生成和识别需要OpenAI API
   - Ollama本地模型暂不支持图像功能

4. **网络连接**：
   - Ollama模式可完全离线使用
   - OpenAI模式需要联网
