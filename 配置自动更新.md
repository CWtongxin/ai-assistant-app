# 🚀 自动更新快速配置指南

## 第一步：创建GitHub仓库

### 1. 访问GitHub
打开浏览器访问：https://github.com/new

### 2. 创建新仓库
填写以下信息：
- **Repository name**: `ai-assistant-app`（或其他你喜欢的名字）
- **Description**: `超级全能AI助手`
- **Public/Private**: 选择 Public（公开）
- 点击 **Create repository**

### 3. 记录仓库信息
创建完成后，记录以下信息：
- **用户名**: 你的GitHub用户名（例如：zhangsan）
- **仓库名**: ai-assistant-app

---

## 第二步：修改配置文件

### 文件1: `electron/main.js`

找到第25-31行，修改为你的GitHub信息：

```javascript
autoUpdater.setFeedURL({
  provider: 'github',
  owner: '你的GitHub用户名',    // ← 改成你的
  repo: 'ai-assistant-app',     // ← 改成你的仓库名
  private: false
})
```

**例如**：如果你的用户名是 `zhangsan`，就改成：
```javascript
autoUpdater.setFeedURL({
  provider: 'github',
  owner: 'zhangsan',
  repo: 'ai-assistant-app',
  private: false
})
```

### 文件2: `package.json`

找到第40-45行，修改为你的GitHub信息：

```json
"publish": {
  "provider": "github",
  "owner": "你的GitHub用户名",    // ← 改成你的
  "repo": "ai-assistant-app",     // ← 改成你的仓库名
  "releaseType": "release"
}
```

---

## 第三步：打包应用

```bash
npm run electron:build:win
```

打包完成后，在 `release` 目录会生成：
- `AI插件应用 Setup 1.0.0.exe` - 安装包
- `latest.yml` - 更新配置文件（重要！）

---

## 第四步：发布到GitHub

### 方法一：使用GitHub网页（推荐）

1. **访问Releases页面**
   ```
   https://github.com/你的用户名/ai-assistant-app/releases
   ```

2. **点击 "Create a new release"**

3. **填写发布信息**
   - **Tag version**: `v1.0.0`
   - **Release title**: `v1.0.0 - 初始版本`
   - **Description**: 
     ```
     超级全能AI助手 v1.0.0
     
     功能特性：
     - 支持Ollama本地模型
     - 支持OpenAI API
     - 黑白极简设计
     - 完全中文界面
     ```

4. **上传文件**
   点击 "Attach binaries by dropping them here or selecting them."
   上传以下两个文件：
   - `AI插件应用 Setup 1.0.0.exe`
   - `latest.yml`

5. **点击 "Publish release"**

### 方法二：使用GitHub CLI（如果已安装）

```bash
# 登录GitHub
gh auth login

# 创建Release
gh release create v1.0.0 \
  --title "v1.0.0 - 初始版本" \
  --notes "超级全能AI助手初始版本" \
  release/"AI插件应用 Setup 1.0.0.exe" \
  release/latest.yml
```

## 第五步：测试自动更新

### 1. 安装应用
在另一台电脑上安装 `AI插件应用 Setup 1.0.0.exe`

### 2. 启动应用
应用会自动检查更新（但因为是最新版本，所以不会提示）

### 3. 发布新版本测试更新
```bash

# 2. 重新打包
npm run electron:build:win

# 3. 创建新Release
# Tag: v1.0.1
# 上传新文件

# 4. 等待几分钟，已安装的应用会自动检测到更新
```

## ⚠️ 常见问题

### Q1: 提示"无法连接到更新服务器"
**原因**：GitHub配置不正确
**解决**：
1. 检查 `owner` 是否是你的GitHub用户名
2. 检查 `repo` 是否是仓库名（不是完整URL）
3. 确认Release已发布

### Q2: 没有收到更新通知
**原因**：版本号没有更新或Release未发布
**解决**：
1. 确认 `package.json` 中的 `version` 已更新
2. 确认GitHub Release的Tag与版本号一致
3. 确认已上传 `latest.yml` 文件

### Q3: 开发模式能测试更新吗？
**不能**。开发模式下自动更新被禁用。
**解决**：需要打包后才能测试。
