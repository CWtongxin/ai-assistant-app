# 🚀 快速开始 - 3步完成GitHub自动更新

## 当前状态
✅ 所有代码配置已完成！  
📦 v1.0.0 已打包完成！  
🎯 只需发布到GitHub即可启用自动更新！

---

## 📝 第一步：发布v1.0.0到GitHub

### 运行发布脚本
```powershell
cd e:\AI插件
.\发布.ps1 -Version "1.0.0" -Title "v1.0.0 - 初始版本"
```

### 按提示操作
脚本会引导你：
1. ✅ 检查文件是否存在
2. 🌐 自动打开GitHub Release页面
3. 📤 提示你上传文件

### 需要上传的文件
在GitHub Release页面上传：
- 📦 `release/AI插件应用 Setup 1.0.0.exe`
- 📄 `release/latest.yml` ⚠️ **必须上传！**
- 📄 `release/AI插件应用 Setup 1.0.0.exe.blockmap`

### 填写Release信息
- **Tag version**: `v1.0.0`
- **Release title**: `v1.0.0 - 初始版本`
- **Description**: 简单介绍你的应用功能

点击 **"Publish release"** ✅

---

## 🎉 第二步：测试自动更新

### 安装应用
1. 从GitHub下载 `AI插件应用 Setup 1.0.0.exe`
2. 安装到电脑
3. 启动应用

### 发布新版本测试
```powershell
# 1. 更新版本号
npm version patch  # 1.0.0 -> 1.0.1

# 2. 重新打包
npm run electron:build:win

# 3. 发布新版本
.\发布.ps1 -Version "1.0.1" -Title "v1.0.1 - 测试更新"
```

### 验证更新
打开已安装的v1.0.0应用，等待几秒应该会收到更新通知！

---

## 🔄 第三步：日常发布流程

以后每次发布新版本：

```powershell
# 1. 更新代码...

# 2. 更新版本号
npm version patch  # 或小改minor，大改major

# 3. 打包
npm run electron:build:win

# 4. 发布
.\发布.ps1 -Version "新版本号" -Title "版本说明"
```

**用户会自动收到更新！** 🎊

## ⚠️ 重要提醒

1. **latest.yml 必须上传** - 没有它自动更新会失败
2. **版本号必须匹配** - package.json的version和Release的Tag要一致
3. **开发模式不测试更新** - 必须打包后才能测试
